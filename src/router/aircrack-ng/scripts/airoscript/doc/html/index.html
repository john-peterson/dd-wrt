DESCRIPTION

Airoscript is program designed to ease the use of aircrack-ng.
Airoscript allows you to scan select and attack any detected access poing
and guides you trought different kinds of attacks available for selected access point's encryption type.


SYNOPSIS

airoscript   [options]


OPTIONS

TERMINAL
Especifies the terminal to use instead of the one in config file.

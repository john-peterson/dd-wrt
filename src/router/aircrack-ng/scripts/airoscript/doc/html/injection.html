Injection

There are two kind of injections possible: Frag injection and chopchop injection.
Both of them can be called with or without clients.
Injecting requires having a xor file, so a chopchop or frag attack must have been done before injecting.
Airoscript calls arpforge and then aireplay ( with injectrate especified in airoscript.conf).

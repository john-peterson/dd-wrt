Deauthentication

It can deauth everybody, fake mac and selected client.
For selected client you have to select a client first.
It launches aireplay-ng with --deauth option and deauth time set in airoscript.conf.
If option 1 is selected, it won't use the -a option, otherwise it will use -a FAKE_MAC or -a CLIENT_MAC.
